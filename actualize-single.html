Actualize CRM

Customer Relationship Management application that allows Actualize to keep track of its clients' activities. This application makes extensive use of the Twilio API in order to contact both new and existing clients.

Technologies used:

  • eRuby backend
  • Javascript VueJS frontend
  • PostgreSQL database
  • Twilio API
